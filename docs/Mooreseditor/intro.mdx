# Introduction
[mooreseditor](https://editor.moores.tech/)はブラウザベースのmod開発ツールです。
誰でも簡単にいい感じのmodを開発、公開できるようにすることを目的に開発されています。  
現在はアルファ版で機能も最低限ですが、今後機能を追加していきます。

## 対応状況
現在は**PC版**の以下のブラウザのみ対応しています。
- Chrome
- Edge
- Opera

:::danger 以下のブラウザは対応していません

- Firefox
- Safari(プレビュー版を除く)
未対応ブラウザはエディタ内で使用している[File System Access API](https://developer.mozilla.org/en-US/docs/Web/API/File_System_Access_API)が対応していないためです。
対応予定は今の所ありませんので、ご了承ください。

:::




## 現在の機能
- メタデータの編集
- アイテム、ブロック、鉱石、クラフトレシピ、機械レシピの追加、編集機能

## 今後の機能
- C#コードの編集、追加機能
- ビジュアルスクリプティング機能
- アイコン用画像生成機能
- 3Dモデル、音声、画像などのアセット追加、ロード機能
- 簡易モデリングツールや簡易ペイントツール
